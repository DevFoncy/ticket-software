Install project , requirements : 
   .Laravel v >5.6
   .PHP v >7.2
   .MYSQL
   .composer 
   
 Steps: 
   1) git clone URL name
   2) cd name
   3) composer install
   4) php artisan key:generate
   5) php artisan migrate:refresh --seed
   6) php artisan passport:install
   7) php artisan serve
